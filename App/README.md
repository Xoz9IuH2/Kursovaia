# ВОЕНКОМАТ24

Веб-приложение для военного комиссариата.

## Функциональность

### Роли:
- **Призывник** - просмотр повесток, подача заявлений, запись на приём
- **Сотрудник** - личные дела, повестки, заявления, календарь мероприятий
- **Комиссар** - сотрудники, статистика, отчёты, аудит

## Запуск

```bash
# Установить зависимости
npm install

# Запустить бэкенд (терминал 1)
node server.js

# Запустить фронтенд (терминал 2)  
npm run dev
```

## Демо-аккаунты

| Логин | Пароль | Роль |
|-------|--------|------|
| commissar | 123 | Комиссар |
| employee1 | 123 | Сотрудник |
| citizen1 | 123 | Призывник |

## Стек

- React + Vite
- Express
- SQLite (better-sqlite3)

Безопасность и доступ (для сотрудников)
- Вход в приложение сейчас ограничен только для сотрудников военкомата (роли employee и commissar).
- Логин получает JWT (JSON Web Token). Все дальнейшие запросы к API требуют заголовок Authorization: Bearer <token>.
- Взаимодействие для призывников будет реализовано позже в мобильном приложении.
- Эндпоинты регистрации/создания пользователей доступны только сотрудникам (RBAC).
